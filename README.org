*** Opening nvim
On the project directory containing the .godot file, open nvim with the following command:
#+BEGIN_EXAMPLE
nvim --listen ./godothost .
#+END_EXAMPLE
It's good to setup an alias to open nvim this way, I have one in my bashrc called `gvim`
